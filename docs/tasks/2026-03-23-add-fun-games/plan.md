# 小游戏合集新增4个经典游戏 实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**目标：** 在 `games.html` 中新增贪吃蛇、迷你扫雷、跳跳小人（跑酷）、弹球（打砖块）四个经典且好玩的游戏。

**架构：** 沿用现有 tab 切换架构，每个游戏一个 `init` 函数 + 对应 CSS。所有游戏基于 Canvas 绘制（高性能、流畅动画）。新增4个 tab 按钮，`loadGame()` 分发到对应初始化函数。

**技术栈：** HTML5 Canvas + 原生 JavaScript + CSS（无依赖）

## 现有代码结构

- **文件**: `games.html`（单文件，内联 CSS + JS）
- **现有游戏**: 记忆翻牌、反应测试、接Emoji、猜数字、颜色记忆、情话打字（共6个）
- **模式**: tab 按钮切换 → `loadGame()` 调用对应 `initXxx()` → 在 `#gameArea` div 中渲染

## 新增游戏设计

### ① 🐍 贪吃蛇

| 项目 | 说明 |
|------|------|
| **玩法** | 方向键/虚拟按钮控制蛇移动，吃食物变长，碰墙或自身结束 |
| **画布** | 300×300 Canvas，15×15 格子 |
| **操作** | 键盘方向键 + 页面下方4个虚拟方向按钮（手机友好） |
| **记分** | 吃一个食物+1分，localStorage 记录最高分 |
| **速度** | 初始 150ms/步，每吃5个加速 10ms，最快 60ms |
| **食物** | 随机 emoji（🍎🍕🍩🍓🧁🍪） |

### ② 💣 迷你扫雷

| 项目 | 说明 |
|------|------|
| **玩法** | 8×8 棋盘，10颗雷。左键翻开/点击翻开，长按或右键标旗 |
| **画布** | Grid 布局（非 Canvas），每格 36×36px |
| **操作** | 点击翻开，长按 500ms 标旗（手机友好） |
| **胜利** | 翻开所有非雷格子 |
| **数字** | 周围雷数用不同颜色显示（1蓝/2绿/3红/4紫） |

### ③ 🏃 跳跳小人

| 项目 | 说明 |
|------|------|
| **玩法** | 小人自动向右跑，点击屏幕/空格键跳跃躲避障碍物 |
| **画布** | 全宽 Canvas，高 200px |
| **操作** | 点击画布或按空格键跳跃 |
| **难度** | 障碍物间隔随分数递减（越快越难），速度逐渐加快 |
| **元素** | 小人用 emoji 🏃，障碍物随机（🌵🪨📦🗑️），地面线 |
| **记分** | 存活时间得分，localStorage 记录最高 |

### ④ 🏓 弹球打砖块

| 项目 | 说明 |
|------|------|
| **玩法** | 经典 Breakout，控制底部挡板反弹球打碎上方砖块 |
| **画布** | 全宽 Canvas，高 360px |
| **操作** | 鼠标/触摸移动控制挡板位置，点击发球 |
| **砖块** | 5行8列彩色砖块，不同行不同颜色 |
| **球** | 白色圆球，碰墙/砖块/挡板反弹 |
| **生命** | 3条命，球掉下去扣一条，全没了游戏结束 |
| **记分** | 每块砖1分，localStorage 记录最高分 |

## 任务分解

### Task 1: 新增 CSS 样式

**文件:** 修改 `games.html` 的 `<style>` 部分

添加以下 CSS：
- 蛇方向按钮样式（`.snake-controls`、`.d-btn`）
- 扫雷棋盘样式（`.mine-grid`、`.mine-cell`）
- Canvas 通用样式（`.game-canvas` 居中、响应式宽度）
- 弹球挡板提示样式

### Task 2: 添加 Tab 按钮和 loadGame 分发

**文件:** 修改 `games.html` 的 HTML 和 JS 部分

- 在 `.game-tabs` 中新增4个 `<button>`
- 在 `loadGame()` 中新增4个 `else if` 分支

### Task 3: 实现贪吃蛇游戏

**文件:** 修改 `games.html` 的 `<script>` 部分

核心逻辑：
1. `initSnake()` - 创建 Canvas + 分数栏 + 虚拟方向键
2. `snakeLoop()` - 游戏主循环（setInterval）
3. `drawSnake()` - 绘制蛇身和食物
4. 键盘和虚拟按钮事件监听
5. 碰撞检测（墙壁 + 自身）

### Task 4: 实现扫雷游戏

**文件:** 修改 `games.html` 的 `<script>` 部分

核心逻辑：
1. `initMine()` - 生成棋盘 HTML
2. `placeMines()` - 随机布雷 + 计算周围雷数
3. `revealCell()` - 翻开格子（空白递归展开）
4. `toggleFlag()` - 标旗（长按检测）
5. 胜负判定

### Task 5: 实现跳跳小人游戏

**文件:** 修改 `games.html` 的 `<script>` 部分

核心逻辑：
1. `initRunner()` - 创建 Canvas + 分数栏
2. `runnerLoop()` - requestAnimationFrame 主循环
3. 重力跳跃系统（跳跃力 + 重力加速度）
4. 障碍物生成和滚动
5. 碰撞检测（矩形碰撞）

### Task 6: 实现弹球打砖块游戏

**文件:** 修改 `games.html` 的 `<script>` 部分

核心逻辑：
1. `initBreakout()` - 创建 Canvas + 分数/生命栏
2. `breakoutLoop()` - requestAnimationFrame 主循环
3. 球的运动 + 碰壁反弹
4. 砖块碰撞检测 + 消除
5. 挡板跟随鼠标/触摸
6. 发球/失球/生命管理

### Task 7: 整合测试和提交

- 在浏览器中逐个测试全部10个游戏
- 确认移动端虚拟按钮可用
